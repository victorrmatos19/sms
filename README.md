# ERP Desktop

Sistema ERP Desktop para pequenas empresas brasileiras.

## Requisitos de desenvolvimento

- Java 21 LTS
- Maven 3.9+
- IntelliJ IDEA (recomendado)

## Como rodar em desenvolvimento

```bash
mvn javafx:run
```

## Banco de dados

PostgreSQL embarcado — sobe automaticamente com a aplicação.
Os dados ficam em: ~/erp-desktop/data/
